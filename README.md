# Machine-Learning-Foundations


This repository contains a collection of foundational implementations of core machine learning algorithms, developed as part of my training in data science and quantitative methods.
The goal is to build a solid understanding of model mechanics, optimization, and evaluation, which are essential for more advanced applications in quantitative finance.
